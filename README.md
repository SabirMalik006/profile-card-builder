# Profile Card Builder

A simple frontend interview task where users can fill a form and generate a live profile card preview.

## How to Run
Open `index.html` directly in the browser.

## Tech Used
- HTML5
- CSS3
- Vanilla JavaScript
- React (ProfileCard component)

## Features
- Form validation without alert
- Live card preview update
- Dynamic skill tags rendering
- Responsive layout with media queries

## Time Taken
~40-50 minutes

- Better UI styling and animations
